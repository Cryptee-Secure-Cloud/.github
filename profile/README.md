## What is Cryptee?

Cryptee is a privacy-focused, client-side encrypted, and cross-platform productivity suite and data storage service [citation:3]. It provides a secure, encrypted home on the internet for your photos, videos, documents, notes, and journals [citation:6]. Founded in 2017 by John Ozbay in Estonia, Cryptee is designed for journalists, reporters, lawyers, psychotherapists, and individuals who have a greater need for privacy and security for their digital belongings [citation:2][citation:3].

Infrastructure teams managing private data benefit from Cryptee's zero-knowledge architecture—your files are encrypted on your device before they ever leave it, and your encryption keys are never sent to Cryptee's servers [citation:5][citation:10]. This means no one, not even Cryptee staff, can access your documents or photos [citation:10].

## Key Features

| Feature | Description |
|---------|-------------|
| **Client-Side Encryption** | All data is encrypted using AES-256 before leaving your device. It would take a supercomputer millions of years to crack open your data [citation:4][citation:5]. |
| **Zero-Knowledge Architecture** | Your encryption keys are never sent to Cryptee's servers—we can't read your documents or see your photos [citation:10]. |
| **Ghost Folders & Ghost Albums** | Hide sensitive folders and albums. They only become visible if you know their exact name—providing ultimate deniability under coercion [citation:3][citation:6]. |
| **Open Source Code** | Cryptee's source code is publicly available for anyone to audit, verify, and identify vulnerabilities [citation:3][citation:4]. |
| **Anonymous Sign-Up** | No email or personal information required—just a username and password [citation:4][citation:5]. |
| **Cross-Platform Access** | Works on Windows, Mac, Linux, iOS, Android, and even e-readers with a browser [citation:1][citation:6]. |
| **Rich Document Editing** | Markdown support, KaTeX math, to-do lists, code highlighting, tables, PDF export, and DOCX editing [citation:3][citation:6]. |
| **Photo Organization** | Albums, tags (#hashtags), typo-tolerant search, descriptions, EXIF data, and RAW file support [citation:4]. |
| **Secure Album Sharing** | Share encrypted albums with end-to-end encrypted links [citation:4]. |
| **Evernote Import** | Built-in support for bulk import from Evernote notes [citation:3][citation:4]. |
| **Offline Mode** | Works even when you're offline—install as a Progressive Web App (PWA) [citation:1][citation:4]. |
| **GDPR Compliant** | Data protection by design and default—no cookies, no trackers [citation:4]. |
| **Based in Estonia** | Outside 14-eyes jurisdictions, protected from mass surveillance and corporate espionage [citation:4][citation:5]. |

## Installation Guide

### Installation on Windows

Cryptee is a Progressive Web App (PWA)—you can use it directly in your browser or install it as a desktop app on Windows [citation:1].

**Option 1: Use in Browser (No Installation)**

**Step 1:** Open the Cryptee website at `https://crypt.ee` using Vivaldi, Firefox, Edge, or Google Chrome [citation:1].

**Step 2:** Create an account—no email required, just a username and password [citation:4].

**Step 3:** Start using Cryptee Docs and Photos immediately.

**Option 2: Install as Desktop App (PWA)**

**Step 1:** Open `https://crypt.ee` using Vivaldi, Google Chrome, or Microsoft Edge [citation:1].

**Step 2:** Click the install button in the address bar (right side):
- **Microsoft Edge:** Click the install icon in the address bar
- **Google Chrome:** Click the install icon in the address bar
- **Vivaldi:** Right-click on the Cryptee tab and select "Install Cryptee" [citation:1]

**Step 3:** The app will install as a proper desktop app that works even when you're offline [citation:1].

### Why is Cryptee Not in App Stores?

Cryptee is not available on traditional app stores for six main reasons [citation:1]:

1. **Avoiding Monopoly Control** — App stores are run by tech monopolies that can censor or remove apps on behalf of foreign governments.
2. **Fast Updates** — App stores can delay updates—critical in security and privacy contexts.
3. **App Store Tax** — 30% payment commission is financially and privacy-wise absurd.
4. **Deniability** — App stores leave traces of downloads, making deniability harder for at-risk users.
5. **Practicality** — All features work via web technologies, making native apps unnecessary.
6. **Speed** — Install directly from the browser with one click.
